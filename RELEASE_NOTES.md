# SoftDo v1.6.0 - Release Notes

This version represents the first major feature-complete release of SoftDo.

## Highlights

- **Elegant Identity**: New minimalist "S" logo designed for high-end desktop aesthetics.
- **Glassmorphism UI**: Beautiful transparent interface with customizable opacity.
- **Auto-Update**: Built-in update checker linked directly to GitHub.
- **Desktop Ready**: Advanced tray integration, always-on-top mode, and startup support.

## Full Evolution History (v1.0.0 - v1.6.0)

### v1.6.0 (Latest)

- **New Brand Identity**: Switched to a premium white Squircle logo with black typography.
- **UI Refinement**: Fixed context menu and button clipping (specifically the "Clear All" button).
- **Icon Assets**: Optimized all system icons for high-DPI Windows displays.

### v1.3.0 - v1.5.0

- **Translation Layer**: Multi-language support (English/Chinese).
- **Notification System**: Preparation for smart due-time alerts.
- **Stability**: Fixed Electron main process IPC deadlocks.

### v1.2.6

- **UI Polish**: Unified button styles with Apple-like squircle rounding.
- **Time Picker**: Improved layout spacing and button visibility.
- **Animations**: Fixed "teleporting" empty state animation.
- **Layout**: Fixed layout shift caused by scrollbar behavior.

### v1.2.3 - v1.2.5

- **Task Management**: Added in-place renaming and detail editing.
- **Auto-Update**: Initial implementation of the update fetcher.
- **Tray Fixes**: Switched tray icon to PNG to resolve white background issues.

### v1.0.0 - v1.1.0

- **Foundation**: Initial Glassmorphism todo framework.
- **Core Controls**: Window management, pinning, and opacity sliders.
