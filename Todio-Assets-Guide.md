# Todio 应用界面生图提示

## 应用说明

**Todio** 是一个桌面待办事项管理应用（桌面小组件），用于帮助用户管理日常任务。应用采用无边框窗口设计，支持透明度调节，可以置顶显示在桌面上。

**主题模式**：应用支持**浅色模式（Light Mode）**、**深色模式（Dark Mode）**和**玻璃主题（Glass Mode）**三种主题，用户可以在设置中切换。所有界面都需要生成三种主题模式的版本（Light / Dark / Glass）。

## 生图提示词

### 中文提示词

> 设计一个桌面待办事项管理应用 "Todio" 的完整界面。这是一个无边框的桌面小组件窗口，垂直布局。界面包含：顶部标题栏（左侧有 Logo、应用名和版本号，右侧有设置、置顶、最小化、关闭四个按钮），主内容区（标题"今日任务"和任务数量、清空按钮、任务输入框和设置提醒按钮及添加按钮、任务列表），以及各种弹窗（设置弹窗、时间选择器、任务操作菜单、任务详情编辑、时间修改等）。界面设计要清晰展示所有功能按钮和交互元素，不需要写具体的样式细节，只需要完整展示所有页面和功能按钮的位置和布局。

### 英文提示词

> Design the complete interface for a desktop to-do management app called "Todio". It's a frameless desktop widget window with a vertical layout. The interface includes: a top title bar (logo, app name and version on the left, settings, pin, minimize, and close buttons on the right), main content area (title "Today's Tasks" with task count, clear all button, task input box with set reminder button and add button, task list), and various popups (settings popup, time picker, task action menu, task details editor, time modification, etc.). The interface design should clearly show all functional buttons and interactive elements, without specific style details, just complete display of all pages and the position and layout of all functional buttons.

## 跨平台与主题规范

### 1. 统一设计规范（Unified Electron）
- **标题栏**：自定义统一标题栏（忽略 Windows/macOS 原生差异）
  - 左侧：Logo + 应用名称 + 版本号
  - 右侧：设置按钮（齿轮）| 置顶按钮（图钉）| 最小化 | 关闭
- **窗口控制**：
  - 可拖拽区域：整个标题栏 + 内容区空白处
  - 不可拖拽：按钮、输入框等交互元素
- **圆角**：
  - 窗口：8px 圆角
  - 按钮/输入框：6px 圆角
  - 卡片/弹窗：12px 圆角

### 2. 主题模式说明
所有界面需要生成以下三种主题版本：
- **浅色模式（Light Mode）**
  - 背景：`#F8FAFC`
  - 文字：`#1E293B`
  - 强调色：`#7C3AED`（紫色）
  - 玻璃效果：`rgba(255, 255, 255, 0.8)`
  - 阴影：浅灰色微阴影

- **深色模式（Dark Mode）**
  - 背景：`#0F172A`
  - 文字：`#F1F5F9`
  - 强调色：`#A78BFA`（浅紫色）
  - 玻璃效果：`rgba(15, 23, 42, 0.7)`
  - 发光：紫色微光

- **玻璃主题（Glass Mode）**
  - 背景：`rgba(255, 255, 255, 0.15)`
  - 文字：白色 + 70% 透明度
  - 强调色：`#C4B5FD`（浅紫）
  - 毛玻璃效果：`backdrop-filter: blur(20px)`
  - 边框：`rgba(255, 255, 255, 0.1)`

## 需要生成的界面场景

**注意**：每个界面场景都需要生成**浅色模式**、**深色模式**和**玻璃主题**三个版本。

2. **设置弹窗打开状态**（显示所有设置选项和开关）
   - 浅色模式版本
   - 深色模式版本

3. **时间选择器打开状态**（显示时间选择器和 AM/PM 切换）
   - 浅色模式版本（背景更不透明，白色背景）
   - 深色模式版本

4. **任务操作菜单打开状态**（显示所有菜单选项）
   - 浅色模式版本
   - 深色模式版本

5. **任务详情编辑弹窗打开状态**（显示所有编辑字段和按钮）
   - 浅色模式版本（白色玻璃效果，更不透明）
   - 深色模式版本

6. **时间修改弹窗打开状态**（显示时钟和快速调整按钮）
   - 浅色模式版本（渐变紫色背景，白色玻璃面板）
   - 深色模式版本

7. **空状态界面**（显示空状态提示和按钮）
   - 浅色模式版本（浅色背景，白色玻璃卡片）
   - 深色模式版本

8. **欢迎提示显示状态**（显示欢迎信息和浮动提示框）
   - 浅色模式版本
   - 深色模式版本

9. **更新提示显示状态**（显示更新通知卡片和统计信息）
   - 浅色模式版本（渐变背景的更新通知卡片）
   - 深色模式版本

### 浅色模式特点

- 背景：浅色背景（#F3F4F6、#F8FAFC 等）
- 玻璃效果：白色半透明背景，更不透明（rgba(255, 255, 255, 0.65-0.85)）
- 文字：深色文字（#1e293b、#334155 等）
- 边框：白色边框，更明显
- 阴影：柔和的阴影效果

### 深色模式特点

- 背景：深色背景（#0F172A、#12122A 等）
- 玻璃效果：深色半透明背景（rgba(15, 23, 42, 0.6) 等）
- 文字：浅色文字（白色、浅灰色）
- 边框：半透明白色边框，较淡
- 阴影：较深的阴影效果
